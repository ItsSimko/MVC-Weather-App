 A simple package providing the \href{http://www.google.com/fonts/specimen/Roboto}{\texttt{ Roboto}} fontface. The font was created by \href{https://plus.google.com/110879635926653430880/about}{\texttt{ Christian Robertson}}.\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_84616d56d7d8c824dfa9a50be072a878_autotoc_md4309}{}\doxysection{\texorpdfstring{Installing}{Installing}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_84616d56d7d8c824dfa9a50be072a878_autotoc_md4309}
Assuming you have \href{http://nodejs.org/}{\texttt{ Node\+JS}}, \href{https://www.npmjs.com/}{\texttt{ NPM}} and \href{http://bower.io/}{\texttt{ Bower}} installed globally just open up a terminal, navigate to your projects root directory and then execute


\begin{DoxyCode}{0}
\DoxyCodeLine{\#\ install\ via\ NPM}
\DoxyCodeLine{\$\ npm\ install\ roboto-\/fontface\ -\/-\/save}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ install\ via\ Bower}
\DoxyCodeLine{\$\ bower\ install\ roboto-\/fontface\ -\/-\/save}

\end{DoxyCode}

There\textquotesingle{}re several files in the {\ttfamily css/} subdirectory. Import them in your project to have access to "{}\+Roboto"{} font face\+:


\begin{DoxyItemize}
\item {\ttfamily css/roboto/roboto-\/fontface.\+css} -\/ whole font family compiled to CSS
\item {\ttfamily css/roboto/sass/roboto-\/fontface.\+scss} -\/ whole font family in SCSS
\item {\ttfamily css/roboto/less/roboto-\/fontface.\+less} -\/ whole font family in LESS
\item {\ttfamily css/roboto-\/condensed/roboto-\/condensed-\/fontface.\+css} -\/ whole font family compiled to CSS
\item {\ttfamily css/roboto-\/condensed/sass/roboto-\/condensed-\/fontface.\+scss} -\/ whole font family in SCSS
\item {\ttfamily css/roboto-\/condensed/less/roboto-\/condensed-\/fontface.\+less} -\/ whole font family in LESS
\item {\ttfamily css/roboto-\/slab/roboto-\/slab-\/fontface.\+css} -\/ whole font family compiled to CSS
\item {\ttfamily css/roboto-\/slab/sass/roboto-\/slab-\/fontface.\+scss} -\/ whole font family in SCSS
\item {\ttfamily css/roboto-\/slab/less/roboto-\/slab-\/fontface.\+less} -\/ whole font family in LESS
\end{DoxyItemize}

Importing whole family may be unnecessary and lead to huge build, so if you are using SCSS or LESS, you can import only individual weights by importing for example\+:


\begin{DoxyItemize}
\item {\ttfamily css/roboto/sass/roboto-\/fontface-\/black.\+scss}
\item {\ttfamily css/roboto/sass/roboto-\/fontface-\/black-\/italic.\+scss}
\end{DoxyItemize}\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_84616d56d7d8c824dfa9a50be072a878_autotoc_md4311}{}\doxysection{\texorpdfstring{Hinting}{Hinting}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_84616d56d7d8c824dfa9a50be072a878_autotoc_md4311}
