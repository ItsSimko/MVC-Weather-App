Dotenv es un módulo de dependencia cero que carga las variables de entorno desde un archivo {\ttfamily .env} en \href{https://nodejs.org/docs/latest/api/process.html\#process_process_env}{\texttt{ {\ttfamily process.\+env}}}. El almacenamiento de la configuración del entorno separado del código está basado en la metodología \href{http://12factor.net/config}{\texttt{ The Twelve-\/\+Factor App}}.
