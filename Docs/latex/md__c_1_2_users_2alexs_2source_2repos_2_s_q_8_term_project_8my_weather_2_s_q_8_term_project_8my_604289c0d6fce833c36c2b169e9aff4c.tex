 A backwards-\/compatible fs.\+realpath for Node v6 and above

In Node v6, the Java\+Script implementation of fs.\+realpath was replaced with a faster (but less resilient) native implementation. That raises new and platform-\/specific errors and cannot handle long or excessively symlink-\/looping paths.

This module handles those cases by detecting the new errors and falling back to the Java\+Script implementation. On versions of Node prior to v6, it has no effect.\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_604289c0d6fce833c36c2b169e9aff4c_autotoc_md3050}{}\doxysection{\texorpdfstring{USAGE}{USAGE}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_604289c0d6fce833c36c2b169e9aff4c_autotoc_md3050}
