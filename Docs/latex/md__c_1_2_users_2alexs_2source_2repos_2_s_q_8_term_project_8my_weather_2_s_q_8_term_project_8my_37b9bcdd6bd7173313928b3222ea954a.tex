By having only a single instance of these functions around, it\textquotesingle{}s possible to do some nice optimizations. Eg. \href{https://github.com/fb55/CSSselect}{\texttt{ {\ttfamily CSSselect}}} uses these functions to determine whether a selector won\textquotesingle{}t match any elements. If that\textquotesingle{}s the case, the DOM doesn\textquotesingle{}t even have to be touched.\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_37b9bcdd6bd7173313928b3222ea954a_autotoc_md2095}{}\doxysubsubsection{\texorpdfstring{And why is this a separate module?}{And why is this a separate module?}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_37b9bcdd6bd7173313928b3222ea954a_autotoc_md2095}
