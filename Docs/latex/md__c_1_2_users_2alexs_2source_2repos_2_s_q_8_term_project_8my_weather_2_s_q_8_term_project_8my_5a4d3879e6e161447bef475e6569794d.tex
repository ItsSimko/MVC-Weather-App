This config is specifically designed to be used by {\ttfamily @vue/cli} \& {\ttfamily create-\/vue} setups and is not meant for outside use (it can be used but some adaptations on the user side might be needed -\/ for details see the config file).

This ruleset is the base configuration for Vue-\/\+Type\+Script projects. Besides setting the parser and plugin options, it also turns off several conflicting rules in the {\ttfamily eslint\+:recommended} ruleset. So when used alongside other sharable configs, this config should be placed at the end of the {\ttfamily extends} array.

This is extended from the {\ttfamily @typescript-\/eslint/recommended} ruleset, which is an \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+\_\+opinionated\+\_\+\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} ruleset. See the \href{https://github.com/typescript-eslint/typescript-eslint/tree/master/packages/eslint-plugin/src/configs\#recommended}{\texttt{ original documentation}} for more information.

Some of its rules, however, might conflict with {\ttfamily prettier}. So when used alongside other sharable configs, this config should be placed after all other configs except for the one from {\ttfamily @vue/eslint-\/config-\/prettier} or {\ttfamily eslint-\/plugin-\/prettier} in the {\ttfamily extends} array.
