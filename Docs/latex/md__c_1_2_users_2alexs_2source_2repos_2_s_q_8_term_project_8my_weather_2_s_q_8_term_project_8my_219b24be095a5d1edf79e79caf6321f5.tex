Check if the path is executable. If no callback provided, and a global {\ttfamily Promise} object is available, then a Promise will be returned.

Will raise whatever errors may be raised by {\ttfamily fs.\+stat}, unless {\ttfamily options.\+ignore\+Errors} is set to true.\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_219b24be095a5d1edf79e79caf6321f5_autotoc_md3250}{}\doxysubsection{\texorpdfstring{$<$tt$>$isexe.\+sync(path, \mbox{[}options\mbox{]})$<$/tt$>$}{<tt>isexe.sync(path, [options])</tt>}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_219b24be095a5d1edf79e79caf6321f5_autotoc_md3250}
