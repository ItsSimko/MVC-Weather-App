{\bfseries{Webpack Virtual Modules}} is a plugin that allows for dynamical generation of in-\/memory virtual modules for Java\+Script builds created with webpack. When virtual module is created all the parent virtual dirs that lead to the module filename are created too. This plugin supports watch mode meaning any write to a virtual module is seen by webpack as if a real file stored on disk has changed.\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_29235b204b98b37d8ad27e77a2450723_autotoc_md5224}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_29235b204b98b37d8ad27e77a2450723_autotoc_md5224}

You can generate virtual modules \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+\_\+dynamically\+\_\+\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} with Webpack Virtual Modules.

Here\textquotesingle{}s an example of dynamic generation of a module. All you need to do is create new virtual modules using the plugin and add them to the {\ttfamily plugins} array. After that, you need to add a webpack hook. For using hooks, consult \mbox{[}webpack compiler hook documentation\mbox{]}.


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ webpack\ =\ require('webpack');}
\DoxyCodeLine{var\ VirtualModulesPlugin\ =\ require('webpack-\/virtual-\/modules');}
\DoxyCodeLine{}
\DoxyCodeLine{//\ Create\ an\ empty\ set\ of\ virtual\ modules}
\DoxyCodeLine{const\ virtualModules\ =\ new\ VirtualModulesPlugin();}
\DoxyCodeLine{}
\DoxyCodeLine{var\ compiler\ =\ webpack(\{}
\DoxyCodeLine{\ \ //\ ...}
\DoxyCodeLine{\ \ plugins:\ [}
\DoxyCodeLine{\ \ \ \ virtualModules}
\DoxyCodeLine{\ \ ]}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{compiler.hooks.compilation.tap('MyPlugin',\ function(compilation)\ \{}
\DoxyCodeLine{\ \ virtualModules.writeModule('node\_modules/module-\/foo.js',\ '');}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{compiler.watch();}

\end{DoxyCode}
