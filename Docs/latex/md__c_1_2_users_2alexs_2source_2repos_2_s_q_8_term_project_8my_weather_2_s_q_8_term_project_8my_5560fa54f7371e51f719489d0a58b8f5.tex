The ESLint custom parser for {\ttfamily .vue} files.\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_5560fa54f7371e51f719489d0a58b8f5_autotoc_md5145}{}\doxysection{\texorpdfstring{‚§¥Ô∏è Motivation}{‚§¥Ô∏è Motivation}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_5560fa54f7371e51f719489d0a58b8f5_autotoc_md5145}
This parser allows us to lint the {\ttfamily \texorpdfstring{$<$}{<}template\texorpdfstring{$>$}{>}} of {\ttfamily .vue} files. We can make mistakes easily on {\ttfamily \texorpdfstring{$<$}{<}template\texorpdfstring{$>$}{>}} if we use complex directives and expressions in the template. This parser and the rules of \href{https://github.com/vuejs/eslint-plugin-vue}{\texttt{ eslint-\/plugin-\/vue}} would catch some of the mistakes.\hypertarget{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_5560fa54f7371e51f719489d0a58b8f5_autotoc_md5146}{}\doxysection{\texorpdfstring{üíø Installation}{üíø Installation}}\label{md__c_1_2_users_2alexs_2source_2repos_2_s_q_8_term_project_8my_weather_2_s_q_8_term_project_8my_5560fa54f7371e51f719489d0a58b8f5_autotoc_md5146}
