Composition API in Vue 2 is provided as a plugin and needs to be installed on the Vue instance before using. Normally, {\ttfamily vue-\/demi} will try to install it automatically. For some usages where you might need to ensure the plugin gets installed correctly, the {\ttfamily install()} API is exposed to as a safe version of {\ttfamily Vue.\+use(\+Composition\+API)}. {\ttfamily install()} in the Vue 3 environment will be an empty function (no-\/op).
